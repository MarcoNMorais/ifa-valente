PROJETO IFA VALENTE — INÍCIO RÁPIDO

1. Instale o Python 3.11 ou superior no computador.
2. Dê dois cliques no arquivo iniciar.bat.
3. Na primeira execução, aguarde a instalação automática das dependências.
4. O sistema abrirá em: http://127.0.0.1:5000/ifa

UNIDADES CADASTRADAS
O banco já contém as 10 Unidades de Saúde informadas para o município de Valente.

DADOS INICIAIS
O banco inicia sem profissionais e sem avaliações cadastradas.

IMPORTANTE
Troque as senhas e configure uma SECRET_KEY segura antes do uso oficial.
Para hospedagem, use armazenamento persistente para o banco de dados e os backups.

Consulte o arquivo README.md para instruções completas.
