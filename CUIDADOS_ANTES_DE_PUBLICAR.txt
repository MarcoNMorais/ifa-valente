# CUIDADOS ANTES DE USAR ONLINE

1. Trocar a senha do ADM.
2. Trocar a senha do Regulador.
3. Não apagar o disco persistente no Render.
4. Não apagar a pasta data se estiver usando localmente.
5. Fazer backup manual antes de alterações grandes.
6. Se for publicar para uso oficial, testar:
   - Login ADM
   - Login Regulador
   - Cadastro de ACS
   - Nova avaliação
   - Regras dos índices 2 e 3
   - Férias/licença
   - Relatório por ACS
   - Exportação CSV
   - Backup
